# Aplicacoes-em-C-2012
Estudos da época da faculdade
Repositório com aplicações simples desenvolvidas por volta de 2012, no início da graduação.

Os códigos refletem estágio inicial de aprendizado, podendo conter limitações técnicas, ausência de boas práticas e soluções simplificadas. Foram mantidos sem alterações para preservar o contexto original de desenvolvimento.

Inclui exemplos como estruturas de dados (pilha, árvore), operações matemáticas básicas, manipulação de entrada/saída e pequenos testes de lógica.

Este repositório não tem como foco qualidade de código atual, mas sim documentação histórica.
